Featured Projects

Hubble Space Telescope Data Pipeline

Reduced and aligned HST WFC3/UVIS images of the globular cluster NGC 6440. Performed distortion correction, PSF photometry, and astrometric calibration using Python and Fortran. Produced high-precision stellar catalogs and proper-motion analysis to search for black hole candidates.

📁 Code Repository

Machine Learning for Exoplanet Transit Classification

Trained SVM and neural network models to detect exoplanetary transits in simulated and real datasets. Implemented data preprocessing, cross-validation, and error analysis in Python (scikit-learn, NumPy, Matplotlib).

📁 Code Repository

Exoplanet Transit Analysis: WASP‑12b

End‑to‑end data reduction of space‑ and ground‑based observations for WASP‑12b. Calibrated raw CCD data, performed aperture photometry, and built differential light curves. Used Bayesian MCMC (emcee) to extract planetary parameters, integrating TESS and TASTE datasets.

📁 Code Repository

Globular Cluster Morphology with Gaia DR3

Processed Gaia DR3 data to quantify ellipticities of 126 globular clusters. Wrote custom Python scripts for cross‑matching, data visualization, and statistical analysis. Presented results on how these clusters are not as spherical as traditionally thought.

📁 Code Repository

High‑Energy Astrophysics: M51 with Swift

Reduced UV and X‑ray data of the interacting galaxy system M51 using HEASoft and XSPEC. Produced co‑added images, light curves, and calibrated spectra. Modeled the X‑ray spectrum with absorbed power‑law fits to investigate AGN activity.
