Featured Projects

🔭 Hubble Space Telescope Data Pipeline – NGC 6440

HST WFC3/UVIS · PSF Photometry · Proper Motions

Developed a complete reduction pipeline for HST images of globular cluster NGC 6440. Performed distortion correction, astrometric alignment, and PSF photometry using Python and Fortran routines. Produced high-precision stellar catalogs and conducted proper-motion analysis to identify high-velocity star candidates and measure velocity dispersion profiles, contributing to dynamical studies of dense stellar systems.

🪐 Machine Learning for Exoplanet Transit Classification

SVM · Neural Networks · Feature Engineering

Built and compared ML classifiers (SVM and shallow neural networks) to identify exoplanet transit signals in synthetic and real light curves. Implemented data preprocessing, feature extraction from folded transit profiles, cross-validation, and ROC analysis. Achieved >92% classification accuracy on Kepler candidate data, demonstrating how ML accelerates transit detection in large survey data.

scikit-learn TensorFlow NumPy

📉 Exoplanet Transit Analysis: WASP‑12b

TESS + TASTE · Aperture Photometry · MCMC Modeling

Performed end-to-end transit analysis for the ultra-hot Jupiter WASP-12b. Reduced TESS and ground-based TASTE data, extracted differential photometry, and modeled transits using Bayesian MCMC (emcee) to refine orbital and planetary parameters. Combined multi-epoch datasets to improve precision and assess timing variations.

🌌 Globular Cluster Morphology with Gaia DR3

Astrometry · Ellipticity Measurement · Large-Scale Catalog Analysis

Analyzed Gaia DR3 data for 126 globular clusters, computing ellipticities and orientation angles using custom Python pipelines. Cross-matched cluster members, applied statistical cleaning, and visualized spatial distributions. Findings challenge the traditional spherical model, revealing significant flattening in the outer regions.

Gaia DR3 Astroquery Matplotlib

🛰️ Vegetation Response to Extreme Rainfall in Arid Muscat (Sentinel-2)

Earth Observation · NDVI · Remote Sensing

Led a remote sensing case study after a record rainfall event (March 2026) in Muscat, Oman. Processed Sentinel-2 L2A imagery (10m resolution) pre- and post-event using Python (rasterio, numpy). Quantified NDVI changes: only 0.3% of the area showed greening, with a mean ΔNDVI = +0.004, highlighting the challenges of arid ecosystem monitoring. This work demonstrates practical EO skills applicable to climate resilience and land cover change detection.
